Formularios HTML5

Posibles valores para type

  • text Para introducir texto, simplemente.
  • search Específico para input en un buscador.
  • url Específico para direcciones de página web. Requiere http:// o http:
  • tel Para números de teléfono.
  • email Para direcciones de correo electrónico. Requiere algo@algo.
  • color Muestra una paleta de colores a elegir.
  • password Escribe el contenido en asteriscos.
