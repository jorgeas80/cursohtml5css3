Selectores contextuales

  • Vecino adyacente Selecciona todos los elementos p que estén justo a continuación de elementos h2 
    h2 + p {
    background-color: red
}
  • Vecino Selecciona todos los elementos p que sean vecinos de elementos h2 
    h2 ~ p {
    background-color: red
}
