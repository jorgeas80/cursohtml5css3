Selectores CSS3

Pseudo-clases: aplicables a elementos adyacentes (II)

  • :nth-child Selecciona el elemento de clase foo cuando es el hijo en posición n, de todos los hijos de su padre 
    .foo:nth-child(n) {
    background-color: green;
}
  • :only-child Selecciona el elemento de clase foo cuando es el único hijo de su padre 
    .foo:only-child {
    background-color: green;
}
