Selectores CSS3

Selectores de atributos (II)

  • Empieza por Selecciona los elementos de tipo párrafo cuyo atributo id empieza por "id_" 
    p[id^="id_"] {
    background-color: red;
}
  • Termina por Selecciona los elementos de tipo párrafo cuyo atributo id termina por "_excel" 
    p[id$="_excel"] {
    background-color: green;
}
