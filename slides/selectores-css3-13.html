Selectores CSS3

Selectores de atributos (I)

  • Presente Selecciona los elementos de tipo párrafo que tengán presente el atributo id 
    p[id] {
    background-color: red;
}
  • Exacto Selecciona los elementos de tipo párrafo cuyo atributo foo vale bar 
    p[foo="bar"] {
    background-color: blue;
}
