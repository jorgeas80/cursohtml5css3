Formularios HTML5

Más posibles valores para type

  • datetime, date, month, week, time, datetimelocal Para marcas temporales.
  • number Específico para números
  • range Rango numérico, representado por una barra. Requiere min y max.
  • checkbox Para seleccionar o deseleccionar.
  • radio Para elegir entre diferentes opciones..
  • file Permite subir ficheros. Acepta un atributo accept, para especificar tipos MIME aceptados. Requiere formenctype=multipart/form-data
